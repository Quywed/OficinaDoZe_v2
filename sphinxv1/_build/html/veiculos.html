veiculos module

veiculos.insert_veiculo(conn)[código fonte]

Insere um veículo na tabela “Veiculos” da base de dados.

Parâmetros:

conn (sqlite3.Connection) – Conexão ativa com o base de dados SQLite.

Levanta:

ValueError – Se o ano estiver fora do intervalo válido (1900-2023) ou se a quilometragem for negativa.

veiculos.list_veiculos(conn)[código fonte]

Lista todos os veículos presentes na tabela “Veiculos” da base de dados.

Parâmetros:

conn (sqlite3.Connection) – Conexão ativa com a base de dados SQLite.

veiculos.select_veiculo_cliente(conn, client_nif)[código fonte]

Seleciona os veículos de um cliente específico da tabela “Veiculos”.

Parâmetros:

  • conn (sqlite3.Connection) – Conexão ativa com a base de dados SQLite.

  • client_nif (int) – NIF do cliente para o qual os veículos serão selecionados.
